# DSKG
A Deep Sequential Model for Knowledge Graph Completion

### Requirements
* python 3.x
* tensorflow 1.x
* numpy, pandas
* jupyter

### Running

1. unpack the data.tar.gz, which concludes all of three datasets.

2. run jupyter:
<code> jupyter notebook </code>

3. open runDSKG.ipynb & run all cells (Kernel -> Restart & Run All)
